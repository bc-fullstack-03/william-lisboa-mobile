## Features

- Register User
- Login User
- Logout Account
- Publish Post
- Follow Friends
- Like Post
- Comment Post
- Cross platform

- React-native
- Expo
- Axios
- jwt-decode
- phosphor-react-native
- expo-image-picker
- typescript
